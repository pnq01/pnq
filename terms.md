# 项目术语词汇表（中英文对照）by301潘能琦

| 英文术语                            | 中文翻译                                     | 说明                                                                                                                               |
| ------------------------------- | ---------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| FastAPI                         | FastAPI                                  | 一个现代、快速（高性能）的 Web 框架，用于构建基于 Python 的 API。它基于标准的 Python 类型提示，利用 Starlette 和 Pydantic 实现快速开发，具备自动生成交互式 API 文档、支持异步编程等特性。           |
| JWT (JSON Web Token)            | JSON Web 令牌                              | 一种用于用户身份验证的令牌格式，可实现无状态认证机制。它由三部分组成：头部（Header）、负载（Payload）和签名（Signature），常用于在客户端和服务器之间安全地传输信息，尤其适用于分布式系统的身份验证和授权。                 |
| Middleware                      | 中间件                                      | 在请求和响应之间处理逻辑的组件，例如日志记录、权限验证、CORS 处理等。中间件可以对进入的请求进行预处理，也可以对即将返回的响应进行后处理，是一种在不修改业务逻辑核心代码的情况下增强功能的有效方式。                             |
| Dependency Injection            | 依赖注入                                     | 一种设计模式，用于自动传入控制器或函数所需的依赖组件。通过依赖注入，可以降低代码之间的耦合度，提高代码的可测试性和可维护性，使得代码更易于扩展和修改。                                                      |
| Endpoint                        | 接口端点                                     | API 中暴露给客户端访问的 URL 路径，如 /login 或 /posts/{id}。每个端点对应一个特定的 API 功能，客户端可以通过发送 HTTP 请求到这些端点来调用相应的服务。                                  |
| OAuth2                          | OAuth2 授权协议                              | 一种开放的授权标准，FastAPI 中常用于实现基于令牌的认证。它允许用户在不向第三方应用提供其用户名和密码的情况下，授权第三方应用访问其受保护的资源，提供了一种安全、灵活的授权机制。                                     |
| ORM (Object Relational Mapping) | 对象关系映射                                   | 用于数据库操作的一种技术，如 SQLAlchemy，将数据库表结构映射为 Python 类对象。通过 ORM，开发者可以使用面向对象的方式操作数据库，避免直接编写复杂的 SQL 语句，提高开发效率和代码的可维护性。                      |
| Schema (Pydantic)               | 数据模型（Pydantic）                           | 使用 Pydantic 定义的请求 / 响应数据格式和验证规则。Pydantic 基于 Python 类型提示，能够自动对输入的数据进行验证和转换，确保数据的有效性和一致性，同时还能生成详细的错误信息。                            |
| Route / Router                  | 路由 / 路由器                                 | 定义 URL 请求与函数处理之间的映射关系。路由用于将客户端的请求分发到相应的处理函数，而路由器则是对路由进行组织和管理的组件，使得代码结构更加清晰和模块化。                                                  |
| Environment Variable            | 环境变量                                     | 用于存储如数据库连接地址、密钥等配置信息，通常存放在 .env 文件中。环境变量可以根据不同的运行环境进行配置，避免将敏感信息硬编码在代码中，提高了应用的安全性和可配置性。                                           |
| aiosqlite                       | 异步 SQLite 数据库驱动                          | 一个基于 SQLite 的异步数据库驱动，允许在异步 Python 代码中使用 SQLite 数据库。它利用 Python 的异步特性，避免在数据库操作时阻塞事件循环，提高了程序的并发性能。                                  |
| alembic                         | 数据库迁移工具（支持 SQLAlchemy）                   | 一个专门为 SQLAlchemy 设计的数据库迁移工具，用于管理数据库模式的版本控制。它允许开发者在不丢失数据的情况下对数据库结构进行修改，同时可以记录和回滚数据库的变更历史。                                         |
| annotated-types                 | 带注释的类型工具（用于类型提示增强）                       | 提供了一种在 Python 类型提示中添加额外元数据的方式，增强了类型提示的表达能力。可以使用这些注释来提供更多关于类型的信息，如验证规则、默认值等，有助于提高代码的可读性和可维护性。                                     |

| asyncpg                         | 异步 PostgreSQL 数据库驱动                      | 一个用于与 PostgreSQL 数据库进行异步交互的 Python 驱动。它充分利用 Python 的异步特性，在处理大量并发数据库请求时表现出色，能够显著提高应用程序的性能。                                        |
